# RolText M3 Design Assistant

This agent is built to support design-focused development in the RolText project.

## Primary goals
- Analyze and improve the user interface using Material Design 3 principles.
- Help scaffold workflows, tasks, and documentation that reflect the current project structure.
- Use markdown files as knowledge sources for project context and design decisions.
- Keep enhancements compatible with the existing HTML and JavaScript architecture.

## When using this agent
1. Read `PROJECT-OVERVIEW.md` to understand the repo scope and design intent.
2. Inspect `index.html`, `styles-m3.css`, and relevant `js/` modules for the current UI flow.
3. Propose or implement changes that preserve the theme, readability, and game UX.
4. Document any new patterns or design systems in markdown.
